Cara Export/Import Animasi Grease Pencil ke Krita

ditulis oleh Sadewoo
  • Software yang dipakai Blender v4.5 and Krita v5
  • File dan Addon/Plugin yang perlu di download & install terlebih dahulu :
    ╰┈➤SakugaGP Addon (Install di Blender) : Direct Gumroad
    ╰┈➤Camera Overscan Addon (Install di Blender) : Direct
    ╰┈➤SakugaKR (Install di Krita) : Direct
    ╰┈➤File Blender yang berisi template animasi Krita : Direct
  • Buka file Blender templatenya
  • Di addon SakugaGP klik tombol ini, tombol ini untuk mengganti ukuran canvas (Camera) menjadi full
  • Setelah itu Collection "Template" nya di "Exclude from view layer" (uncheck checkbox yang ada di gambar bawah) biar templatenya tidak ikut ter-render
  • Kemudian di addon SakugaGP, Klik Render > "Transparent" dan "Set to PNG"
  • Viewportnya akan tampak seperti ini :
  • Kemudian di addon SakugaGP, Klik Render > Pastikan tipe nama filenya (Name) adalah "Frame Numbers" dan klik "Render All"
  • Blender akan nge-freeze sementara karena sedang me-render semua keyframe nya, harap tunggu sampai proses selesai
  • Folder "export" akan otomatis di buat di dalam folder yang sama dengan project file blender
  • Jika proses render selesai, Buka Krita dan buat file baru dengan template animasi
  • Tampilkan plugin SakugaKR nya di Settings > Docker > Cek "SakugaKR", lalu akan muncul docker seperti ini :
  • Di sebelah teks "Path" tersebut di ganti dengan path folder "export" yang tadi
  • Klik "Create Layers" , Layer animasi kosong akan otomatis terbuat sesuai nama-nama folder yang ada di dalam folder "export" itu
  • Klik salah satu Layer animasi kosong itu, klau klik "Import Images", tunggu sampai gambar-gambar nya ter-import
  • Ulangi langkah 11 tapi untuk layer kosong lainnya
  • Ubah durasi awal animasi menjadi 1 (Clip Start = 1) dan akhir animasi menjadi 24 (Clip End = 24)
  • Selesai!
